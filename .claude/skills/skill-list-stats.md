# Статистика подписчиков SendPulse

Когда пользователь спрашивает про статистику подписчиков:

1. Вызови `get_mailing_lists` чтобы получить все списки рассылки.

2. Для каждого списка покажи:
   - Название
   - Всего подписчиков (all_email_qty)
   - Активных (active_email_qty)
   - Неактивных (inactive_email_qty)
   - Дата создания

3. Подведи итоги:
   - Общее количество подписчиков по всем спискам
   - Процент активных
   - Самый большой список
   - Самый новый список

4. Предложи дальнейшие действия: посмотреть статистику кампаний или отправить рассылку.
